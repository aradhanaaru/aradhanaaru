## 🐍 Snake Eats My Contributions

<div align="center">

> Animated in hot pink & electric purple. **Needs a one-time setup** — see step-by-step guide below, or the full walkthrough in `SETUP.md`.

<details>
<summary>⚡ Quick setup (click to expand)</summary>

1. Create the file `.github/workflows/snake.yml` in your `aradhanaaru/aradhanaaru` repo and paste in the contents of **SNAKE_WORKFLOW.yml** (provided alongside this README).
2. Commit and push it to your `main` branch.
3. Go to the **Actions** tab of that repo → select **generate animation** → click **Run workflow**.
4. Wait ~30 seconds. GitHub will auto-create an `output` branch containing the pink/purple snake SVG — that's what the image above points to.
5. It regenerates automatically every day after that. No further edits needed.

Full details, troubleshooting, and the Matrix-rain asset setup are in **SETUP.md**.
</details>
